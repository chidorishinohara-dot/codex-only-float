# Contributing to codex-only-float

Thank you for your interest in contributing to codex-only-float. This project is an early-stage open-source Windows utility for monitoring Codex session and weekly usage in a lightweight floating window.

## How to contribute

You can help by:

1. Reporting bugs.
2. Improving documentation.
3. Suggesting Windows usability improvements.
4. Testing the app in different Windows environments.
5. Improving error handling and release packaging.

## Reporting issues

When opening an issue, please include:

1. Your Windows version.
2. How you ran the project.
3. What you expected to happen.
4. What actually happened.
5. Screenshots or logs if available.

Please do not include private credentials, tokens, cookies, or personal account data in issues.

## Pull requests

Before opening a pull request:

1. Keep changes focused and small.
2. Explain what problem the PR solves.
3. Add or update documentation when behavior changes.
4. Avoid adding credential storage or unsafe data collection.
5. Make sure the app remains privacy-focused.

## Project direction

The project should stay lightweight, local-first, and transparent. The goal is to help Codex users monitor usage more conveniently without unnecessary complexity.
